Summary

Steps for creating and installing SCST DKMS Debian package on Ubuntu 15.04-17.04+.

This has been tested to work on VirtualBox VM's for Ubuntu 15.04-17.04.

This is a WIP and this page may be updated frequently.

Supported Distros

Currently supported and tested distros:

Ubuntu 15.04 Vivid Vervet

Ubuntu 15.10 Wily Werewolf

Ubuntu 16.04 Xenial Xerus *

Ubuntu 16.10 Yakkety Yak

Ubuntu 17.04 Zesty Zapus

Debian 9 Stretch

Debian 8 Jessie *

* 16.04 4.4.0-x-generic kernels are excluded and will not work with DKMS SCST build. You must upgrade 16.04 to the HWE 4.10.x kernel before running the script (script prompts for permission to automatically install HWE this upgrade for you).

* Debian 8 Jessie SCST DKMS works (create-scst-install.sh) and SAN creation script works (create-scst-oracle.sh) but create-scst-multipath.sh does not create multipath links in /dev/mapper although multipath devices in /dev/ are created.

How to Install SCST DKMS on Ubuntu 15.04-17.04+ & Debian 9

To completely build, install and configure SCST DKMS modules on Ubuntu 15.04-17.04+ do the following steps.

  1. BE SURE you have a working internet connection on the box where SCST will be installed.
  2. Make sure your current user is a member of sudo ALL group ("adduser username sudo") as root
  3. If you had to add current username to sudo then logout and login again
  4. Download the GitHub zip archive to the Ubuntu host of the Orabuntu-LXC project here
  5. Unzip the archive
  6. Navigate to "orabuntu-lxc-master/orabuntu/archives" directory
  7. Untar the scst-files.tar archive
  8. chmod +x *.sh vercomp
  9. Run ./create-scst.sh
  10. (Optional) Note: 4K/4K logical/physical LUNs can be configured in create-scst-oracle.sh optionally

What the Script Does

The script will do the following steps automatically:

1. Get the SCST source code from svn

2. Get the zip file of the github.com/gstanden/scst-3.x-debian archive

3. Build the SCST DKMS packages

4. Install the SCST DKMS packages

5. Complete post-install steps to complete the configuration of SCST

6. Complete all configuration of target, group, LUNs, multipath, UDEV rules, on-boot iscsi logins, etc.

As stated above, use the scst-files.tar archive at the Orabuntu-LXC github to fully-automate installing SCST DKMS on Ubuntu 15.04-17.04+.
